Day 9 – Skill Extraction Engine

Objective:
To build a skill extraction engine that extracts technical, business, creative, and soft skills from resume text.

Implementation Details:
- Created a master skill dictionary categorized by skill type.
- Implemented synonym handling (e.g., ML → Machine Learning).
- Implemented stack handling (e.g., MERN → MongoDB, Express, React, NodeJS).
- Built confidence scoring logic based on frequency.
- Removed duplicate skills.
- Generated structured JSON output.

Output:
System returns categorized skills with confidence scores.
